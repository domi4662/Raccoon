== Description ==

Plugin to add team, testimonial and portfolio custom post type. Each post type has its widget and shortcode to use in theme. This addon is best to enhance features of themes as it is easy to use and highly optimized.

Find Demo [Here](https://promenadethemes.com/pt-theme-addon) 

= Team Shortcodes =

[ptta-team]
[ptta-team category="cat-id"]
[ptta-team post_number="x"] where x is positive number
[ptta-team column="x"] where x can be 2,3 or 4
[ptta-team image_size="size"] where size can be thumbnail, medium, large or any custom size defined by theme

= Portfolio Shortcodes =
[ptta-portfolio]
[ptta-portfolio category="cat-id"]
[ptta-portfolio post_number="x"]
[ptta-portfolio column="x"]
[ptta-portfolio image_size="size"]

= Testimonial Shortcodes =
[ptta-testimonials]
[ptta-testimonials category="cat-id"]
[ptta-testimonials post_number="x"]


== Installation ==

1. Go to Plugins > Add New and search for PT Theme Addon. Install and activate it.
2. We are good to go. No configuration needed.


== Frequently Asked Questions ==

= Do this plugin work for other themes ? =

This plugin adds custom post types, meta fields and widgets. These elements can be used in other themes as well.

= Where can I find demo and documentation of plugin ? =

You can find demo and documentation of plugin [here](https://promenadethemes.com/pt-theme-addon)
